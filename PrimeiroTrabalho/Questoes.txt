Exercícios

1) Partição. Uma partição de um conjunto X é qualquer coleção P de subconjuntos não vazios de X tais que todo elemento de X pertença a um e apenas um dos elementos de P. Por exemplo, { {1,3}, {2,4,7}, {5}, {6,8} } é uma partição de {1,2,3,4,5,6,7,8}. O conjunto {6,8} é um dos blocos da partição. Escreva um programa que imprima uma lista de todas as partições de um conjunto {1,2,3,...,n}.

% ./mypart n > arquivo.txt

2) Utilizando a técnica de automatos finitos, implemente um validador de endereço de emails. Monte um script para testar 10 arquivos com 100 emails cada um. Crie os arquivos com emails válidos e inválidos. Gere um arquivo .txt com a saída.

% ./valida_email arquivo-emails1.txt > saida.txt
